Bootstrap Custom SCSS Template

To use this template you can modify any SCSS in the custom.scss file. To compile the modified SCSS with the Bootstrap SCSS you can run 'npm run build-css' in the root of this package. When you run this command all the SCSS will be output to the style.css file.

Replace the scss folder in bootstrap/scss with the current download from bootstrap. Remember this is the in the source download, not the compiled download.
